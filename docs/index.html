IntelliJ Rest CLI is a free commandline application created by QuangSon. It supports to execute IntelliJ HTTP Client file in the commandline. . The application is open source, you can view the source code on github.com/quangson91/intellij_rest_cli.

FEATURES

  • Execute Intellij HTTP request files
  • Running test script
  • Loading and inject environment variables
  • Cross platform: Windows, macOS and Linux
  • Easy to custom via commandline arguments
